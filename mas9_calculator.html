1 Base install — required first
MAS Core is always installed first (holds the user registry). It carries no AppPoint charge on its own.
At MAS 9.1, either the foundation service or Maximo Manage must be present before any other app. The foundation service is a license-restricted version of Manage (user management, security, config). Maximo Manage carries a 300-AppPoint minimum (50 reserved for users).
